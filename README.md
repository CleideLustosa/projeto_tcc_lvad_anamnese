# Interface Web para Anamnese de Pacientes com LVAD

Este projeto faz parte do **Trabalho de Conclusão de Curso (TCC)** desenvolvido para o **colocar nome do projeto Coração**. O objetivo é criar uma interface otimizada para a coleta de dados e monitoramento clínico de pacientes que utilizam Dispositivos de Assistência Ventricular Esquerda (LVAD).

## 🩺 Sobre o Projeto
O uso de dispositivos LVAD exige um acompanhamento rigoroso. Esta aplicação web visa facilitar o processo de anamnese hospitalar, garantindo que informações críticas (dados pessoais, histórico clínico, medicamentos e parâmetros do dispositivo) sejam registradas de forma clara e intuitiva.

### 🚀 Funcionalidades Implementadas (Aba Paciente)
- [x] Cabeçalho com identidade visual institucional.
- [x] Navegação por abas para fluxo de trabalho clínico.
- [x] Sidebar com foto do paciente e histórico de consultas.
- [x] Formulário estruturado de Dados Pessoais e Contatos de Emergência.
- [x] Layout responsivo utilizando Tailwind CSS.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando tecnologias modernas de front-end para garantir performance e usabilidade:
* **React.js**: Biblioteca principal para construção da interface.
* **Tailwind CSS**: Framework para estilização rápida e responsiva.
* **Node.js v24.13.0**: Ambiente de execução do servidor de desenvolvimento.
